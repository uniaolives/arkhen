## 🚀 Quick Start (Local Node)

**Prerequisites:** Docker/Podman, Node.js, Rust.

1. **Install Node dependencies:**
   ```bash
   npm install
   ```
2. **Build and Run the Dashboard:**
   ```bash
   npm run dev
   ```
3. **Deploy the Blockchain Layer:**
   ```bash
   cd arkhe-os-genesis-v1.0
   ./install.sh
   ```
